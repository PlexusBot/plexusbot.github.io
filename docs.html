Plexus Documentation

Standard command and help documentation

Command Usage
plexus help Links to this page
plexus web Links to the official Plexus website
calc [expression] Evaluates basic arithmetic using Pemdas format
plexus xkcd Grabs a random comic strip from xkcd
plexus games Sends a list of games Plexus can host
plexus flip Flips a coin
plexus source Uploads source code for Plexus
plexus dab Sends an image
dance till youre dead Sends a video
